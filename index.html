HTTP

HTTP é a sigla para Hypertext Transfer Protocol, ou Protocolo de Transferência de Hipertexto. Esse é o principal protocolo responsável pela transferência de dados na Internet, criando as bases necessárias para a conexão entre um cliente e um servidor.

HTTPS

HTTPS (Hypertext Transfer Protocol Secure) é uma versão segura do HTTP, que é o protocolo usado para transferir dados na web. A principal diferença é que o HTTPS utiliza criptografia para proteger a comunicação entre o navegador do usuário e o servidor web. Isso é feito através de um protocolo chamado TLS.

DNS

DNS (Domain Name System) é um sistema fundamental para a operação da internet. Ele funciona como uma "agenda telefônica" para a web, traduzindo nomes de domínio amigáveis para endereços IP que os computadores usam para identificar e se comunicar entre si na rede.

FTP

FTP (File Transfer Protocol) é um protocolo padrão utilizado para a transferência de arquivos entre um cliente e um servidor em uma rede, como a Internet.

SSH

O SSH (Secure Shell) é um protocolo de rede usado para acessar e gerenciar sistemas remotos de maneira segura. Ele substitui protocolos mais antigos como o Telnet e o RSH, oferecendo uma camada de criptografia para proteger a comunicação.
